# 📋 Трекер привычек

**Трекер привычек** — мобильное приложение для формирования полезных привычек, разработанное в рамках курсовой работы. Позволяет отслеживать выполнение ежедневных привычек, вести статистику и поддерживать серию дней.

---

## ✨ Основные возможности

- **📋 Управление привычками:** Полный цикл — создание, редактирование, удаление и отметка выполнения
- **🗂 Категории:** Фильтрация привычек по категориям — Здоровье, Спорт, Учёба, Другое
- **📊 Статистика:** График выполнения за 7 дней с цветовой индикацией прогресса
- **🔥 Серия дней (Streak):** Автоматический подсчёт непрерывных дней выполнения
- **🔄 Авто-сброс:** Привычки сбрасываются каждый день в полночь автоматически
- **💾 Локальное хранилище:** Все данные сохраняются на устройстве через AsyncStorage

---

## 🛠 Технологический стек

- **Язык:** JavaScript (ES2021)
- **Фреймворк:** React Native
- **Среда раработки:** Visual Studio Code
- **Инструмент запуска:** Expo
- **Хранилище данных:** AsyncStorage (локальная база данных)
- **Навигация:** Собственная навигация через state

---

## 🚀 Как запустить

1. Установите [Node.js](https://nodejs.org/) и [Expo CLI](https://expo.dev/):
```bash
npm install -g expo-cli
```

2. Склонируйте репозиторий:
```bash
git clone https://github.com/zaynabkerim/habit-tracker.git
cd habit-tracker
```

3. Установите зависимости:
```bash
npm install
```

4. Запустите проект:
```bash
npx expo start
```

5. Откройте приложение Expo Go на телефоне и отсканируйте QR-код.
